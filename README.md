# registas-hub Homebrew Tap

Personal Homebrew tap for [registas-hub](https://github.com/registas-hub) apps.

## Usage

```bash
brew tap registas-hub/tap
brew install --cask gesture-ex
```

## Casks

| Cask | Description |
|------|-------------|
| [gesture-ex](https://github.com/registas-hub/gesture-ex) | macOS right-click on mouse-up + browser mouse gestures |

## Notes

The repository is named `homebrew-tap` on GitHub but referenced as `registas-hub/tap` in `brew tap` (Homebrew strips the `homebrew-` prefix automatically).
